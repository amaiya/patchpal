# MCP (Model Context Protocol) Examples

This directory contains examples and testing scripts for PatchPal's MCP integration.

## Documentation

For complete documentation on MCP support in PatchPal, see:

**[docs/features/mcp.md](../../docs/features/mcp.md)**

The documentation covers:
- Quick start guide
- Configuration (local and remote servers)
- Environment variable expansion
- Finding and managing MCP servers
- In-session commands
- Authentication
- Creating custom servers
- Troubleshooting

## Files in This Directory

- **simple_server.py** - Example local MCP server implementation
- **demo_resources_prompts.py** - Demo of MCP resources and prompts
- **mcp-config.example.json** - Example configuration file
- **test_*.py** - Test scripts for various MCP features
- **TESTING_LOCAL.md** - Guide for local MCP server testing
