# Skills Examples

This directory contains example skills that demonstrate how to create reusable workflows for PatchPal.

## Documentation

For complete documentation on the skills system, see:

**[docs/features/skills.md](../../docs/features/skills.md)**

The documentation covers:
- Creating skills
- Skill file format
- Example skills
- Using skills (direct invocation and natural language)
- Skill priority
- Bundled resources
- Real-world skill ideas
- Skills vs. custom tools

## Example Skills in This Directory

### PatchPal-Created Skills

- **commit/** - Best practices for creating git commits
- **review/** - Comprehensive code review checklist
- **add-tests/** - Add comprehensive pytest tests with templates

- **slack-gif-creator/** - Create animated GIFs for Slack (Apache 2.0, see ATTRIBUTION.md)
